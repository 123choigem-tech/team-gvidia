# 재사용 자산 (assets)

이 폴더에는 다른 작업에서도 바로 복붙해 쓸 수 있는 산출물을 넣는다.

## 현재 포함된 자산
- `reusable_scripts.md`
- `prompt_pack.md`
- `project_notes.md`

## 권장 추가 기준
- 반복 실행 명령
- 데이터 전처리 / 검증 스크립트
- 보고서 생성용 프롬프트
- 팀 공유용 안내문

## 사용 원칙
- 바로 실행 가능한 형태로 적는다.
- 프로젝트 경로를 포함해, 다른 사람이 그대로 따라 할 수 있게 쓴다.
- 실제 반복 작업에 도움이 되는 것만 남긴다.
